
1. Algorithm: Bubble Sort
Best Case Time Complexity: O(n)
Worst Case Time Complexity: O(n²)
Stable: Yes
In-place: Yes
Space Complexity: O(1)

2. Algorithm: Insertion Sort
Best Case Time Complexity: O(n)
Worst Case Time Complexity: O(n²)
Stable: Yes
In-place: Yes
Space Complexity: O(1)

3. Algorithm: Selection Sort
Best Case Time Complexity: O(n²)
Worst Case Time Complexity: O(n²)
Stable: No
In-place: Yes
Space Complexity: O(1)

4. Algorithm: Merge Sort
Best Case Time Complexity: O(n log n)
Worst Case Time Complexity: O(n log n)
Stable: Yes
In-place: No
Space Complexity: O(n)

5. Algorithm: Radix Sort
Best Case Time Complexity: O(nk)
Worst Case Time Complexity: O(nk)
Stable: Yes
In-place: No
Space Complexity: O(n)

6. Algorithm: Counting Sort
Best Case Time Complexity: O(n+k)
Worst Case Time Complexity: O(n+k)
Stable: Yes
In-place: No
Space Complexity: O(n + k)

7. Algorithm: Quick Sort
Best Case Time Complexity: O(n log n)
Worst Case Time Complexity: O(n²)
Stable: Yes
In-place: Yes
Space Complexity: O(log n)
